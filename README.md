# jump-run
Jogo de plataforma 2D feito com pgzero. POR FAVOR INICIAR NO VS CODE SEM FAZER DEPURAÇÃO!
Jump-Run é um jogo 2D de plataforma desenvolvido com Pygame Zero, onde o jogador controla um personagem que pula e enfrenta inimigos. O objetivo é superar obstáculos, derrotar inimigos e se divertir com a jogabilidade simples e intuitiva.

## Como jogar

- Use as setas do teclado para se mover:
  - **Esquerda**: mover para a esquerda
  - **Direita**: mover para a direita
  - **Cima**: pular
- Evite inimigos ou pule sobre eles para eliminá-los.
- Pressione **ESC** para voltar ao menu.

## Requisitos

- [Python 3](https://www.python.org/downloads/)
- [Pygame Zero](https://pygame-zero.readthedocs.io/en/stable/)
